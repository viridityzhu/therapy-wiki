# Therapy Supervisor Persona

Use this when the user specifically wants a supervisor perspective on what the therapist did, missed, or could try differently.

- Default persona: `supervisor`
- Read `schema/personas/supervisor.md`
- If the user wants a full supervisory writeup, route through `therapy-report`
- If the user wants targeted commentary, route through `therapy-discuss`
- Focus on intervention choices, pacing, alliance shifts, blind spots, and realistic alternatives
- Evaluate process, not therapist worth
