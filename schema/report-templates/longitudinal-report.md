# Longitudinal Report Template

Use this structure for a **cross-session Chinese longitudinal report**. The goal is to help the user see what is repeating, what is changing, and what still should not be concluded too early.

## Core stance

- prefer patterns over isolated moments
- resist flattening the whole history into one elegant story
- distinguish stable findings from recent signals
- compare multiple explanations when needed
- keep the report readable enough to revisit later
- write a working map, not a verdict

## Before writing

Make sure you have read enough of:

- `wiki/timeline.md`
- `wiki/formulation/current.md`
- relevant `wiki/themes/*.md`
- relevant `wiki/patterns/*.md`
- the most important session pages behind those summaries

This report should not be written from a report packet alone.

## Recommended structure

### 1. 总体概览

先回答：

- 这批材料里最显眼的 2-4 条主线是什么
- 为什么它们构成了当前这一阶段工作的核心
- 当前最值得继续工作的焦点，和材料刚开始时相比，有没有发生重心变化

### 2. 长期重复主题

按主题写，而不是按 session 罗列。每个主题都说明：

- 它出现在哪些 session
- 它在不同场景里如何换皮不换骨
- 它在关系、工作、家庭、自我评价等场景中的迁移方式

优先选择那些真正跨材料重复出现的主题，而不是一时强烈但证据稀少的主题。

### 3. 重复出现的模式与循环

这里更像在写机制，而不是情节。可以重点组织：

- 情绪循环
- 防御循环
- 关系循环
- 控制与安全感循环
- 自我批评 / 讨好 / 回避 / 脑补等重复结构

如果某个模式只在少数 session 出现，要明确写清它的证据强度。

### 4. 变化、移动与拐点

不要只写“有变化”。要写：

- 什么变了
- 从什么时候开始变
- 哪些材料支持这个判断
- 这种变化更像结构性转变，还是暂时波动
- 是否也出现了反复、退行或假性变化

### 5. 咨询关系与干预的可能作用

尝试回答：

- 哪些类型的干预似乎更有帮助
- 哪些地方长期卡住
- 咨询关系里有没有重复的互动模式
- 来访者在治疗关系中如何测试、依赖、顺从、撤退或对抗

### 6. Competing formulations

至少写出 2 个可能的整体理解版本，并说明：

- 各自解释力在哪里
- 哪些证据更支持当前主版本
- 哪些材料仍让替代版本保持活性

这一节是为了防止报告写成一个过度工整的单一叙事。

### 7. 当前 working formulation

写成“当前最值得工作的地图”，不是终局结论。建议至少包含：

- 核心冲突
- 核心情绪组织方式
- 核心关系期待 / 害怕
- 核心防御与维持循环
- 核心自我叙事
- 当前最可能有效的工作方向

### 8. 仍待验证的问题

这一节非常重要。至少写出：

- 目前证据还不够的判断
- 需要更多 session 才能确认的地方
- 容易被过度解释的部分
- 哪些地方 transcript 质量或材料分布可能影响结论

### 9. 证据附录

按主题或模式列关键片段，而不是乱堆引用。每条引用最好说明：

- 它支持哪一条长期线索
- 它更支持哪一种解释
- 它是否也是反例或限制条件

## Report quality rubric

一份合格的纵向报告，至少应该做到：

- 呈现 3 条以上跨 session 的稳定线索
- 呈现 1-2 个真正的变化或拐点
- 明确至少 2 个“不能过早下结论”的问题
- 既能看到稳定结构，也能看到时间上的移动
- 不把短期状态误写成长期结构
- 不把单次精彩片段误写成总体规律
